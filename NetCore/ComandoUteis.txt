DOT NET ENTITY FRAMEWORK: Instala a biblioteca de intidades relacionais.

dotnet tool install --global dotnet-ef

BUILD: Compila o codigo.

dotnet build

PUBLISH: Gera versão publicável.

donet publish

Release: Gera versão que requer apenas os runtimes.

--configuration Release

SCAFFOLD: Faz uma copia do banco de dado nas aplicação.

dotnet ef dbcontext scaffold "Host=localhost;Database=BibliotecaViva;Username=postgres;Password=example" Npgsql.EntityFrameworkCore.PostgreSQL --output-dir Models\DTO -f

MIGRATTIONS: Atualiza o banco de dados conforme as alterações no código.
