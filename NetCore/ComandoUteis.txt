BUILD: Compila o codigo.

dotnet build

PUBLISH: Gera versão publicável.

donet publish

Release: Gera versão que requer apenas os runtimes.

--configuration Release

SCAFFOLD: Faz uma copia do banco de dado nas aplicação.

dotnet ef dbcontext scaffold "User ID=postgres;Password=example;Server=localhost;Port=5432;Database=BibliotecaViva;Integrated Security=true;Pooling=true;" Microsoft.EntityFrameworkCore.SqlServer --output-dir Model\DAL\Context -f

MIGRATTIONS: Atualiza o banco de dados conforme as alterações no código.
